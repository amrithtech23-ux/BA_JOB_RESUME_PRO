# Business Analyst Job Apply Pro

An ATS-optimized Resume Validator and Generator powered by Streamlit and OpenRouter (Qwen Models).

## Features
1. **Resume Parsing:** Supports PDF, DOCX, and TXT.
2. **JD Validation:** Compares your resume against a Job Description and generates a detailed ASCII report.
3. **ATS Resume Generation:** Rewrites your resume to match the JD using the "Priya Sharma" ATS template structure (Single column, no graphics).
4. **PDF Export:** Downloads the new resume in A4 format.

## Prerequisites
- Python 3.8+
- An OpenRouter API Key (https://openrouter.ai/)
