# Business Analyst Job Apply Pro

An ATS-optimized Resume Validator and Generator powered by Streamlit and OpenRouter (Qwen Models).

![Python Version](https://img.shields.io/badge/python-3.8%2B-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![Streamlit](https://img.shields.io/badge/streamlit-1.31.0-red)

## 🌟 Features

- **Resume Parsing**: Supports PDF, DOCX, and TXT file formats
- **ATS Validation**: Compares your resume against job descriptions with detailed scoring
- **ASCII Report Generation**: Professional validation report with match percentages
- **ATS-Optimized Resume Generation**: Rewrites your resume to match specific job requirements
- **PDF Export**: Downloads the new resume in A4 format with ATS-friendly formatting
- **Qwen Model Integration**: Powered by OpenRouter's Qwen-2.5-72B-Instruct model

## 📋 What It Does

1. **Validates** your current resume against employer job requirements
2. **Generates** a detailed ASCII report showing:
   - Overall eligibility percentage
   - Score breakdown (Education, Skills, Experience)
   - Matched vs. Missing qualifications
   - Personalized recommendations
3. **Creates** an ATS-optimized resume following the Priya Sharma template structure
